## Technologies
- Python
- pandas, seaborn, matplotlib
- scikit-learn

## How to Run
1. Download the dataset and place `kc_house_data.csv` in the same folder.
2. Run `housepriceprediction.ipynb` in Jupyter or Google Colab.
